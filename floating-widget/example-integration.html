Integration Code

To add this widget to your website, simply include this script tag before the closing </body> tag:

<!-- Floating Widget -->
<script src="https://your-widget-domain.vercel.app/widget.js"></script>

Configuration Options

You can configure the widget using data attributes:

  src="https://your-widget-domain.vercel.app/widget.js"
  data-position="bottom-right"
  data-theme="dark"
  data-z-index="1000">
</script>

Testing Different Scenarios

1. Multiple Page Loads

The widget prevents duplicate initialization. Try refreshing the page - you'll only see one widget instance.

2. Dynamic Loading

// Load widget dynamically
const script = document.createElement('script');
script.src = 'https://your-widget-domain.vercel.app/widget.js';
script.async = true;
document.body.appendChild(script);

3. Framework Integration

The widget works with any JavaScript framework (React, Vue, Angular, etc.) as it's completely self-contained.

Note: Replace your-widget-domain.vercel.app with your actual Vercel deployment URL.

Troubleshooting

If the widget doesn't appear:

  1. Check the browser console for errors
  2. Verify the script URL is correct
  3. Ensure your site allows external scripts (check CSP headers)
  4. Try disabling ad blockers
